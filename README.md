# PikaBot-RL
### The Code for rainbow dqn is split into 4 memory, segment tree,rainbowdqn, and ddqnagent it has been implemented in PyTorch and it should run with the latest version of PyTorch once poke env and node 10js have been setup

### The code for reinforce and ppo has been implemented in TensorFlow.It can't be run in the older versions, you would  need Tensorflow 2.10 and Python 3.9 to run the code. Python can be downgraded by creating a conda environment in your cmd and choosing that env as python interpreter(view->command-palette) in vscode.
